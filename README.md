# AI Security Research Lab Ecosystem

Building Trust Infrastructure for Intelligent Systems.

---

## ✅ Overview  
This repository contains the interactive ecosystem map and architecture visualization for our AI Security Research Lab. It is not a generic portfolio or a dashboard, but a living presentation of our operational intelligence modules.

Our ecosystem secures the execution pathway from prompt to action, ensuring that machine intelligence operates transparently, safely, and entirely within the boundaries of human trust.

---

## 🧩 The Ecosystem Architecture

The architecture bifurcates into two main tracks:

### Security Track
Defending the execution pathway through dynamic threat reasoning, cross-module orchestration, and continuous trust propagation.
- **PromptShield**: Every intelligent decision begins with trusted input.
- **SIFTGuardian**: Signals become intelligence when context is verified.
- **Consent Guardian**: Trust becomes enforceable when boundaries are explicit.
- **CodeSage**: Reasoning is valuable only when execution is explainable.
- **SecRitual**: Threat reasoning and execution pathway validation.
- **CyberMesh**: Orchestration of the entire flow of trust across the network.

### Cognitive Track (Human Layer)
Mapping the cognitive boundary and exploring how intelligence safely interacts with human perception, intent, and reality.
- **OpenMind Nexus**: Information Integrity & Trust Formation.
- **NeuroAccess**: Cognitive Security & Human Perception.

## 🛠 Tech Stack  
- React + Vite (TypeScript)
- Raw HTML/CSS & SVG (Architecture Visualizations, Custom Signals & Orbits)
- Intersection Observer (Telemetry Tracker & Scroll Reveal)
- Tailwind CSS (Base structure)

## 🧭 Getting Started  

Clone the ecosystem map:
```bash  
git clone https://github.com/saishamehraa/PortFolio.git  
cd PortFolio  
```

Run the local environment:
```bash
npm i
npm run dev
```
